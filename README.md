
# 🤖 AI Churn Prediction Assistant

An end-to-end Machine Learning application that predicts customer churn and provides actionable business recommendations to improve customer retention.

---

## 🚀 Project Overview

This project helps businesses identify customers who are likely to stop using their services (churn).  
Along with prediction, it also suggests **data-driven retention strategies**, making it useful for real-world business decision-making.

---

## 🎯 Key Features

- 📊 Predicts customer churn probability
- ⚠️ Classifies customers into Low, Medium, High risk
- 💡 Provides smart business recommendations for retention
- 🖥 Interactive UI built using Streamlit
- ⚡ Real-time predictions based on user input

---

## 🛠 Tech Stack

- **Python**
- **Pandas & NumPy**
- **Scikit-learn (ML Model + StandardScaler)**
- **Streamlit (Web App)**
